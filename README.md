# hoppers-ai
Hoppers AI agent, using Minimax with Alpha-Beta Pruning

### Prerequisites
- Python 3
- Numpy

### Installing
1. Create a new virtual env and activate it
```shell
$ python3 -m venv env
```  
```shell
$ source venv/bin/activate
```   
2. Install requirements
```shell
$ pip3 install -r requirements.txt
```   
3. Run `main.py`

- [x] P1 vs. P2
- [x] P vs. Minimax AI Bot
