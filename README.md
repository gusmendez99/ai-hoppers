# hoppers-ai
AI using Minimax with Alpha-Beta Pruning

### Prerequisites
Python 3
Numpy

### Installing
1. Create a new virtual env and activate it
```shell
$ python3 -m venv env
```  
```shell
$ source venv/bin/activate
```   
2. Install requirements
```shell
$ pip3 install -r requirements.txt
```   
3. Run `main.py`
